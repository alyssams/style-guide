Design Architecutre

The goal of the typographic structure is not only to house and contain various levels of information, but also to create appropriate and impactful space around elements within the structure, as well as around the structure as a whole. Each structure featured in this section lends itself to specific formats.

Grids

This is a page layout wireframe illustrating an underlying grid to which all foundational elements within the advertisement align. The number of vertical columns used within the grid varies depending on the dimensions of the format, the amount of content and/or copy, and the amount of desired clear space around the outer edge of the page. The example shown below uses a 13-column grid. Please note the vertical and horizontal relationships of the foundational elements to the columns of the grid.
